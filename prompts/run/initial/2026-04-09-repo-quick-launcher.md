# Prompt: Repo quick-launcher with action menu

Imagine this desired use case. "my weird prompts" is a repo at something like `~/repos/github/project/my-weird-prompts`. I type into an interface (terminal or chatbox): `open my weird prompts`. In response, the CLI finds the repo and then provides me with a numeric list like this:

1. Open in Claude Code
2. Open in File Manager
3. Open in VS Code
4. Open at Terminal
5. Multiplex (Claude, File Manager, Terminal in one tiled window presentation)

What is the easiest way to achieve that? Is there anything existing that bundles all of what's needed, or is this a small custom script? Recommend the simplest route on Ubuntu 25.10 / KDE Plasma Wayland.
